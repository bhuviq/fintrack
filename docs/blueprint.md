# **App Name**: FinTrack

## Core Features:

- Account Summary: Dashboard overview of all accounts, providing a summary of net worth, recent transactions, and financial goals progress.
- Transaction Tracking: Transaction tracking with categorization and tagging. Users can manually add income and expense transactions, categorize them (e.g., groceries, utilities), and add tags for better filtering and analysis.
- AI Transaction Assistant: AI-powered categorization that helps user classify each incoming transaction into corresponding categories. The AI tool will make suggestions for transaction category based on past transactions and user input.
- Budgeting Tools: Budget creation and tracking tools, which enable users to set monthly budgets for different categories and track their spending against these budgets.
- Financial Goals: Goal setting to define financial goals (e.g., saving for a down payment, retirement) and track progress toward those goals.
- Investment Tracking: A read-only connection to brokerage accounts (d-mat) for tracking investment performance.
- Financial Visualization: Visualization and Reporting: Present financial data through charts and graphs to help users understand their spending patterns and investment performance.
- User Management: User login/signup to manage their finances

## Style Guidelines:

- Primary color: A calming blue (#5DADE2) to instill trust and security.
- Background color: A light, desaturated blue (#EBF5FB). This provides a clean and professional backdrop that does not distract from the data.
- Accent color: A complementary orange (#F39C12) for key interactive elements and calls to action.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text to maintain a modern and readable interface.
- Use minimalist and intuitive icons from a consistent set to represent categories, accounts, and actions.
- Emphasize a clean and structured layout, to help guide users, and to focus attention on critical data and actionable items.
- Implement subtle transitions and animations for loading data, updating charts, and confirming user actions, to enhance the user experience without being intrusive.
